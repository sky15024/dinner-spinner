# 🍽️ 今晚吃什麼？ | 文青風晚餐轉盤

這是一個專為選擇困難症設計的網頁小工具，採用文青風的淺綠色調（Sage Green），搭配明體與柔和的低飽和度亮色系轉盤，讓決定晚餐變成一件優雅又有趣的事。

## ✨ 特色功能

- 🎡 **低飽和亮色系轉盤**：柔和的粉、藍、黃、綠、紫配色，搭配深咖啡色文字，優雅且清晰。
- ✏️ **自訂選項**：可自由新增或刪除選項，彈性調整你想抽取的食物。
- 🎲 **隨機範例**：內建多組預設食物類型（日式、台式、西式等），一鍵隨機載入。
- 🎉 **結果彈窗與彩帶特效**：轉出結果時會有與整體風格一致的暖色調碎紙彩帶慶祝。
- 🌿 **優雅文青視覺**：仿紙張材質的背景，搭配精心挑選的墨綠色調與排版。

## 🌐 線上預覽與使用 (GitHub Pages)

您可以透過 GitHub Pages 將此網頁發佈至線上分享給他人：
1. 將本專案推送到您的 GitHub 儲存庫。
2. 進入該儲存庫的 **Settings** -> **Pages**。
3. 在 **Build and deployment** 下的 Source 選擇 **Deploy from a branch**。
4. Branch 選擇 `master` (或 `main`) 以及 `/ (root)`，點擊 **Save**。
5. 等待數分鐘後，即可在 `https://<您的帳號>.github.io/<專案名稱>/` 線上使用！
